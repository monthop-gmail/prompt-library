## บริบทของข้อมูล (Context)

### กิจกรรมที่เก็บข้อมูล

| รายการ | รายละเอียด |

| **เวลา** | 09:00 - 16:00 น. |

## ข้อมูลนี้คืออะไร (What This Data IS)

- ความคิดเห็นของผู้เข้าอบรมที่**เลือกตอบ**แบบสำรวจ
- Snapshot ณ เวลาที่ตอบ (ภายใน 2 วันหลังอบรม)
- Self-reported data (ผู้ตอบประเมินตัวเอง)
- Anonymous (ไม่มีการระบุตัวตน)

---

## ข้อมูลนี้ไม่ใช่อะไร (What This Data IS NOT)

| ไม่ใช่ | เหตุผล |
|--------|--------|
| ความคิดเห็นของพนักงานทั้งบริษัท | เฉพาะคนที่มาอบรมและตอบ survey เท่านั้น |
| การวัดผลการเรียนรู้ | ไม่มี pre-test / post-test |
| การวัด ROI ของการอบรม | ไม่มีการติดตามผลระยะยาว |
| Representative sample | ผู้ตอบเลือกตอบเอง (self-selection bias) |
| ข้อมูลที่เปรียบเทียบกับอบรมครั้งก่อนได้ | นี่คือครั้งแรก ไม่มี baseline |

## ขอบเขตการใช้งาน (Scope of Use)

### ใช้ได้สำหรับ

- รายงานความพึงพอใจเบื้องต้นต่อผู้บริหาร
- ระบุจุดที่ควรปรับปรุงสำหรับการอบรมครั้งต่อไป
- เป็น baseline สำหรับเปรียบเทียบในอนาคต

### ไม่ควรใช้สำหรับ

- ตัดสินประสิทธิภาพของวิทยากรเป็นรายบุคคล
- เปรียบเทียบผลงานระหว่างแผนก
- สรุปว่าพนักงานทั้งบริษัทคิดอย่างไร
- ใช้เป็น KPI ของทีม HR โดยตรง

## ปัญหาคุณภาพข้อมูล (Data Quality Issues)

### ปัญหาที่ทราบแล้ว

| ปัญหา | รายละเอียด | ผลกระทบ | วิธีจัดการ |
|-------|------------|---------|-----------|
| Response rate ต่ำ | 35/47 = 74% | ผู้ไม่ตอบอาจมีความเห็นต่าง | ระบุใน report ว่า n=35 |
| แผนก Marketing น้อย | ตอบ 2 คน จาก 5 คน | ไม่ representative | ไม่แยกวิเคราะห์ Marketing |
| Partial responses | 2 records ไม่ครบ | Missing data | ใช้เฉพาะ complete responses (n=33) |
| Self-selection bias | คนพอใจอาจตอบมากกว่า | อาจ overestimate satisfaction | ระบุข้อจำกัดใน report |

## ข้อจำกัดในการวิเคราะห์ (Analysis Limitations)

### ห้ามทำ

1. **เปรียบเทียบระหว่างแผนก** - บางแผนกมี n < 10
2. **หา correlation กับอายุงาน** - ไม่ได้ตั้ง hypothesis ไว้ก่อน
3. **Sentiment analysis จาก q8_comments** - ไม่มี codebook
4. **สรุปว่าการอบรมสำเร็จ/ล้มเหลว** - ไม่มี success criteria ที่กำหนดไว้ล่วงหน้า

### ควรระวัง

1. Sample size เล็ก → confidence interval กว้าง
2. ครั้งแรก → ไม่มี baseline เปรียบเทียบ
3. Anonymous → ไม่สามารถ follow up ได้
